## Command-line operations

Install dependencies:
```
poetry install
``` 
Start the program:
```
poetry run invoke start
```
Run tests:
```
poetry run invoke test
```
Generate test coverage:
```
poetry run invoke coverage-report
```
_The report will be generated in the htmlcov directory._
